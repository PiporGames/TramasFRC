TramasFRC (nombre genérico) es un proyecto de C++ que consiste en la gestión y envio de tramas de bajo nivel por Ethernet (¡no por WiFi!).
El programa permite al usuario realizar una simulación con otro equipo en la red en la que los equipos escogen diferentes roles en la comunicación para el envio y recepción de tramas.
Existen diferentes modos que el usuario puede seleccionar y diferentes roles, como MAESTRO o ESCLAVO. También puede enviar archivos, carácter por carácter, e incluso crear tramas con datos erroneos.
El objetivo del proyecto es familiarizarse con los aspectos y el funcionamiento del envio y recepción de tramas a bajo nivel de Ethernet.
Este proyecto ha sido creado para la asignatura de Fundamentos De Redes Y Comunicaciones del Grado de Ingeniería en Ingeniería del Software de la UEx año 2023/2024.  

Creado por José Manuel de Torres Domínguez y Manuel Alonso González.  

En este repositorio se encuentra el proyecto de C++.
Para su ejecución es recomendable utilizar VSCode y hacer uso de las configuraciones especiales aplicadas al workspace.
Es necesario tener instalados todos las dependencias que pida el proyecto, evidentemente.
También es recomendable ejecutar el proyecto bajo Linux.
